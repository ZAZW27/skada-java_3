# Tugas 1 java

## Penjelasan: 
Pada kode yang diberikan sudah diberikan contoh penerapan sub-class dalam java yaitu LandVehicle, ia memiliki data uniknya sendiri seperti memilki 'wheels' dan memilki output dari fungsi 'move()' sendirinya.  
Tugas yang harus dilakukuan: 

- Buatlah sub-class baru seperti LandVehicle namun dapat menjadi kendaraan udara, laut, dll. 

- Buat repository baru untuk upload hasil kodingan dengan format nama / title repository: 
    
    ```bash
    java_1-{No. Absen}
    ```
    
  - Lalu menambahkan remote baru di project git yang sama (dimana memiliki remote bernama origin)
    dengan cara
    ```bash
    git remote add <remote_name> <remote_url>
    ```
    
    - <remote_name>:
      
      Dapat berupa nama apa saja, namun tak boleh sama dengan nama remote yang sudah terbuat yaitu "origin". 
      
      contoh: myRepo
    - <remote_url>:
      
      link repository yang telah dibuat di github, biasanya command tersebut sudah tersedia saat membuat repo baru
      
  - Lakukan tracking, committing, dan pushing ke repository:
    ```bash
    git add .
    git commit -m "Membuat sub-class baru"
    git push <remote_name> <branch_name>
    ```

Sehabis selesai mengerjakan tugas, link github dikumpul melalui google form yang telah diberikan.
